| Layer | Function | Description |
| --- | --- | --- |
| Deposit Layer | User deposits & vToken minting | Deposited assets are tokenized into vTokens representing ownership |
| Controller Layer | Strategy execution | Allocates assets across lending, yield strategies, and market operations |
| Oracle Layer | Data feed management | Real-time price, yield, and risk inputs |
| Treasury Router | Revenue routing | Automates buybacks, rewards, and reinvestment |
| Reward Layer | Yield reflection | vToken NAV updates with realized returns |
